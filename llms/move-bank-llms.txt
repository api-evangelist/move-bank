# MOVE Bank

> MOVE Bank (MoveBank Ltd, ABN 91 087 651 090, AFSL/Australian credit licence 234 536) is a 100% customer-owned Australian mutual bank, founded 1968 as Railways Credit Union and rebranded in 2018. As an authorised deposit-taking institution it participates in Australia's Consumer Data Right (CDR / Open Banking) as a data holder, exposing a public, unauthenticated Product Reference Data (PRD) API conforming to the Data Standards Body Consumer Data Standards.

## APIs

- [MOVE Bank CDR Product Reference Data API](https://openbanking.movebank.com.au/OpenBanking): Public, unauthenticated CDS banking Get Products endpoint. Live at https://openbanking.movebank.com.au/OpenBanking/cds-au/v1/banking/products (HTTP 200, x-v 4). Returns a data.products array of banking products (home/personal loans, transaction and savings accounts, term deposits, cards).

## Specs

- [OpenAPI (DSB CDR Banking API v1.36.0)](openapi/move-bank-cds-banking-products-openapi.yml): The shared Consumer Data Standards banking contract the endpoint conforms to. MOVE Bank publicly exposes only the Get Products / Get Product Detail operations; all other operations require the CDR Security Profile.
